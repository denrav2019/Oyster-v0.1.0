== Build Dependencies ==
=============================================================================

To build Oyster from source, the following tools are required:

- GCC (GNU Compiler Collection) or Clang - C compiler (C11 standard)
- GNU Make - Build system
- PCRE2 development libraries (libpcre2-8-dev)

Installation on Ubuntu/Debian:
    sudo apt install build-essential libpcre2-dev

Installation on Fedora/RHEL:
    sudo dnf install gcc make pcre2-devel
