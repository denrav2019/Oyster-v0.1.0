== Special Thanks & Inspiration ==
=============================================================================

The Oyster language was inspired by:

- Perl (Larry Wall)
  The original Perl language that inspired Oyster's philosophy and syntax.
  https://www.perl.org/

- Lua (Roberto Ierusalimschy)
  For the lightweight VM design concepts.
  https://www.lua.org/

- Python (Guido van Rossum)
  For the readable syntax patterns and simplicity philosophy.
  https://www.python.org/

- Rust (Mozilla / Rust Foundation)
  For memory safety concepts and modern tooling ideas.
  https://www.rust-lang.org/
